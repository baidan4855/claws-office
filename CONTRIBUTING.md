# Contributing to Claws Office

感谢你对 Claws Office 项目的兴趣！我们欢迎任何形式的贡献，包括但不限于：

- 🐛 报告 Bug
- 💡 提出新功能建议
- 🔧 提交代码改进
- 📖 完善文档
- 🌐 翻译文档

## 开发环境设置

```bash
# 克隆项目
git clone https://github.com/baidan4855/claws-office.git
cd claws-office

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

## 代码规范

- 使用 TypeScript 进行开发
- 遵循 ESLint 和 Prettier 的代码风格
- 组件采用函数式组件 + Hooks
- 样式使用 CSS Modules 或 CSS-in-JS

## 提交 Pull Request

1. Fork 本仓库
2. 创建你的特性分支 (`git checkout -b feature/amazing-feature`)
3. 提交你的更改 (`git commit -m 'Add some amazing feature'`)
4. 推送到分支 (`git push origin feature/amazing-feature`)
5. 开启 Pull Request

## 问题反馈

如果你发现任何问题，请通过 [GitHub Issues](https://github.com/baidan4855/claws-office/issues) 报告。

# Contributing to Claws Office

Thanks for your interest in contributing to Claws Office! We welcome contributions in various forms:

- 🐛 Bug reports
- 💡 Feature suggestions
- 🔧 Code improvements
- 📖 Documentation
- 🌐 Translations

## Development Setup

```bash
# Clone the repository
git clone https://github.com/baidan4855/claws-office.git
cd claws-office

# Install dependencies
npm install

# Start development server
npm run dev
```

## Code Standards

- Use TypeScript for development
- Follow ESLint and Prettier code style
- Use functional components with Hooks
- Use CSS Modules or CSS-in-JS for styling

## Submitting Pull Requests

1. Fork the repository
2. Create your feature branch (`git checkout -b feature/amazing-feature`)
3. Commit your changes (`git commit -m 'Add some amazing feature'`)
4. Push to the branch (`git push origin feature/amazing-feature`)
5. Open a Pull Request

## Issue Reporting

If you find any issues, please report them via [GitHub Issues](https://github.com/baidan4855/claws-office/issues).
